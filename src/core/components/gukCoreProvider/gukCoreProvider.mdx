<Title>GukCoreProvider</Title>

The `<GukCoreProvider />` is a React context that defines the image, link elements, and copy texts to be rendered across
all the GovKit components, defaulting such values to `<img />`, `<a />` tags, and a default set of copy texts.

The provider becomes particularly useful when an application needs to employ framework-specific images or links, such as
the `<Image />` and `<Link />` components from Next.js, or when specific copy texts need to be customized.

**Usage example**

```typescript
import { GukCoreProvider } from '@aragon/gov-ui-kit';
import NextImage from 'next/image';
import NextLink from 'next/link';

const CustomImage: React.FC<ComponentProps<'img'>> = ({ src = '', alt = 'image', width, height, ...otherProps }) => (
    <NextImage src={src} fill={true} alt={alt} {...otherProps} />
);

const CustomLink: React.FC<ComponentProps<'a'>> = ({ href = {}, ...otherProps }) => (
    <NextLink href={href} {...otherProps} />
);

const coreProviderValues = { Img: CustomImage, Link: CustomLink };

export const Application = () => {
    return (
        <GukCoreProvider values={coreProviderValues}>
            /* Your application */
        </GukCoreProvider>
    )
}
```
